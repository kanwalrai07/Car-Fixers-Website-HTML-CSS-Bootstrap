A responsive, multi-page static website for CARS-FIXERS, a professional auto mechanic and car repair shop located in Karachi, Pakistan. This project showcases the shop's services, gallery, and contact information using modern web design principles.

ğŸ“– Project Overview

CARS-FIXERS is designed to provide an online presence for an automotive repair business. It features a clean, professional layout that allows customers to explore services, view past work in the gallery, and easily get in touch for quotes or appointments.

Pages Included:

Home (home.html): Introduction, key features, vehicle coverage, and testimonials.

Services (services.html): Detailed list of services including Engine Diagnostics, Oil/Lube, Air Conditioning, and more.

About Us (about.html): Information about the company's mission, team, and values.

Gallery (gallery.html): A visual showcase of the workshop and repairs.

Contact (contact.html): A functional layout with a contact form and location details.
